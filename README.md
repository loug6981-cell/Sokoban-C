# Sokoban-C
These are the two code (one for Linux and one for Windows) which are copies of the Sokoban game written in C and run in the terminal.
Download everything and run the program. Once it's done, when the program asks you for the level, enter niveau1.sok, niveau2.sok, etc...

Project completed for a Computer Science degree.

Language: C

Features:
- Player movement
- Block management
- Levels
